How to install
===================

How to install

Proceed the installation using pip :
::
    pip install KernelsPython

Then, import the module "kerpy" in Python :


.. code-block:: python
   :linenos:


    import kerpy
    kerpy.diff.finite().divergence()



Output ::
    | <kerpy.Kernel numpy =
    | [[ 0. -1.  0.]
    | [-1.  2.  0.]
    | [ 0.  0.  0.]] at 0x25fb8477eb0>
